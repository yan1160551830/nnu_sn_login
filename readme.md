# 南京师范大学公共校园网登录
- 开机自动登录
- 登录失败自动重试，直至登录成功

# 使用方法
## 20220119
- 安装node.js环境16版本
- 将程序放到合适的文件夹，之后不能改变文件位置
- 修改config.json中的账户与密码
- 修改nnu_sn_login.bat文件中app.js文件的绝对路径
- 启动start.exe
- C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp
